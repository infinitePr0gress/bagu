# 什么是RAG？描述一下一个完整的RAG系统的详细工作流程？



> RAG系统的工作流程：
>
> 1. query 预处理 （改写、HyDE、多角度扩写），把口语化的问题转成适合检索的形式；
> 2. query Embedding（问题向量化） **注意必须用和建库时相同的模型；**
> 3. 向量检索 + 多路召回  向量检索和 BM25 各有所长，通过 RRF 融合结果；
> 4. Rerank 精排，用 Cross-Encoder 深度理解语义，把粗排的噪音过滤掉；
> 5. Prompt 拼装，明确约束 LLM 只根据资料回答、不知道就说不知道；
> 6. 大模型生成 + 溯源，标注引用来源提升可信度。
